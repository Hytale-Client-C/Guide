# Запуск клиента Hytale

## Описание:

В этом руководстве объясняется, как запустить клиентскую версию Hytale для разработчиков.

## Требуется:

- HytaleClient.exe для разработчиков *(Мы не будем рассказывать вам, где их взять, но все, что попадает в Интернет, остается там навсегда)*
- .NET Framework 4.7
- Visual Studio 2019 или более поздней версии
