# Environment Setup and Project Structure

1. Download and install Visual Studio 2019 or later.
2. Download and install .NET Framework 4.7.
3. Create the following structure *(you can change it, but in our case it was like this)*:

<pre>
release/
├── package/
│   ├── sig
│   ├── jre
│   ├── game
│   │   └── latest
│   └── config.json
└── HytaleClient/
    └── Graphics
        └── Shaders
</pre>

4. Place the `HytaleClient.exe` in the `latest` folder.
